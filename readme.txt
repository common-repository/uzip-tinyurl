== Description ==

Shorten a long url with format http://uzip.net/xxx .

Download it, install it, and active it in your plugin list. You can find it above your TinyMCE Editor - the Rich Editing. 

How to use: Parse a long url in the editor area, select it and click the uzip button. Then it will be turned into a short url soon. Enjoy it.

== Installation ==

1. Upload the files to the directory `/wp-content/plugins/` . it should be like this: /wp-content/plugins/uzip-tinyurl/*
2. Activate the plugin through the 'Plugins' menu in WordPress
